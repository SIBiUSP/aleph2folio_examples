Code in github: aleph2folio

Objetive is convert Aleph data Aleph and import in Folio

Dependencies

Catmandu

This script do:

Foreach record in MARC:

  • convert to FOLIO Codex (using custom Fixes) and create an instance on FOLIO;
  • convert source to MARC-In-JSON and import to source on instance;
  • foreach Library in items, create an Holding and import their items.
This script use CATMANDU that permit to use it with others input formats like MARCXML, ISO2709.
